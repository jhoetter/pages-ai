# @pagesai/hofos-ui

Built hofOS UI surface for PagesAI. Consume it from a GitHub Release tarball:

```json
"@pagesai/hofos-ui": "https://github.com/jhoetter/pages-ai/releases/download/v0.1.0/pagesai-ui-0.1.0.tgz"
```

Run `pnpm --filter @pagesai/hofos-ui build:dist` to produce the library-mode `dist/` artifact.
